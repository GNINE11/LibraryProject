# 📚 Library Project

Projeto básico de livraria online com React e Spring Boot para estudos.

---

## 🛠️ Tecnologias Utilizadas

### 🔹 Frontend

- React 19
- TypeScript
- Vite
- ESLint

### 🔹 Backend

- Java 17
- Spring Boot 3.5.3
  - Spring Web
  - Spring Data JPA
  - Spring Security
- MySQL (banco de dados)
- Maven
